Class Exercises — Bitte schreiben Sie die Sätze

✅ Tip

These sentence patterns come from class notebook exercises. Each picture prompt shows an object, and you practise Ist das ein/eine …? (singular) and Sind das …? (plural).

Pattern: Singular — Ist das ein …? / Nein, das ist kein …

das Auto / der Bus 🚗
Ist das ein Auto? — Is that a car?
Ja, das ist ein Auto. — Yes, that is a car.
Ist das ein Bus? — Is that a bus?
Nein, das ist kein Bus. Das ist ein Auto. — No, that is not a bus. That is a car.
der Mann / die Frau 🧍
Ist das eine Frau? — Is that a woman?
Nein, das ist keine Frau. Das ist ein Mann. — No, that is not a woman. That is a man.
der Computer / das Buch 💻
Ist das ein Buch? — Is that a book?
Nein, das ist kein Buch. Das ist ein Computer. — No, that is not a book. That is a computer.
der Bleistift / die Schere ✏️
Ist das eine Schere? — Is that a pair of scissors?
Nein, das ist keine Schere. Das ist ein Bleistift. — No, that is not scissors. That is a pencil.

Pattern: Plural — Sind das …? / Nein, das sind keine …

die Flaschen 🍾🍾🍾
Sind das Flaschen? — Are those bottles?
Ja, das sind Flaschen. — Yes, those are bottles.
die Flaschen / die Kulis 🍾 / 🖊️
Sind das Kulis? — Are those ballpoint pens?
Nein, das sind keine Kulis. Das sind Flaschen. — No, those are not pens. Those are bottles.
💡 Note

Key grammar note — kein / keine:
Use kein to negate nouns with der or das articles.
Use keine to negate nouns with die articles (including all plurals).
